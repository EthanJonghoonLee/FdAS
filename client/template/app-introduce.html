서울의 발견

[알뜰 서울은요~]

‘알뜰 서울의 발견’은 놓치기 쉬운 서울의 많은 행정서비스를 알뜰하게 이용할 수 있도록 모아 두고 한눈에 볼 수 있도록 한 시민 맞춤 정보 공유 서비스로, 자신의 조건(연령별, 성별 등)에 맞는 맞춤 정보를 제공합니다.
우리 동네 행정 서비스가 궁금하시다고요? 그렇다면 지금 바로 관심 정보를 등록하고 정보를 조회해 보세요. 그런데 이런 내가 알고 있는 서비스가 조회가 안된다구요?
노고팀은 노가다가 최강인 팀이지만 아직 많은 정보를 입력해 나가야 합니다.  본인만 알고 있는 정보가 있다면 알뜰 정보를 기부해 주세요! ‘알뜰 서울의 발견’은 모바일에서도 서비스됩니다. (대단하죠? ^o^)
시민이 알고 있는 행정서비스 데이터를 직접 등록하고, 지인들에게 공유하면서 서비스를 더욱 알뜰하게 만들어 주세요.

[Code for Seoul은 뭐하는 곳 일까요?]

'알뜰 서울의 발견' 서비스 등을 만든 Code for Seoul은 도시사회의 문제를 코드로 해결하기 위해 모인 시민들의 자발적인 모임입니다. Code for Seoul은 도시사회 문제에 관심 있는 시민 이라면 누구나 자유롭게 참여하실 수 있습니다. 매주 월요일 저녁, 핵나잇(Hack Night)에서 밤에 활동하는 슈퍼히어로, 코드포서울 멤버들을 만나실 수 있습니다.
